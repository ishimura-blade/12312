# LadyLife Clinic (static)

Готовый статический сайт для загрузки на любой хостинг.

## Структура

- `index.html` - точка входа (перенаправляет на `clinic.html`)
- `clinic.html` - основная страница клиники
- `assets/images/` - папка для ваших фото мастеров

## Как поставить свои фото

1. Положите изображения в `assets/images/`.
2. Используйте имена:
   - `1.png`
   - `2.png`
   - `3.png`
3. Если имена другие, поменяйте `src` в `clinic.html`.

## Быстрый деплой

### Обычный хостинг (FTP/панель)

1. Загрузите в корень хостинга файлы:
   - `index.html`
   - `clinic.html`
   - папку `assets/`
2. Откройте домен - сайт загрузится.

### GitHub Pages

1. Создайте репозиторий и загрузите эти файлы.
2. В репозитории откройте `Settings -> Pages`.
3. В `Build and deployment` выберите:
   - `Source: Deploy from a branch`
   - `Branch: main` и `/ (root)`
4. Сохраните и подождите 1-2 минуты.
5. Откройте ссылку Pages.
